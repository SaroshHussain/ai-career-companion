# CLAUDE.md

## Project Name

AI Career Companion

## Project Description

AI Career Companion is an AI-powered career assistant that helps users improve their career journey through resume building, interview preparation, cover letter generation, and job application management.

## Tech Stack

Frontend:
- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS

Development Tools:
- Node.js
- Git
- GitHub
- Cursor AI

## Project Structure

- Use a clean and organized folder structure.
- Keep components reusable and modular.
- Separate UI components, pages, and utility functions.
- Follow scalable frontend architecture.

## Coding Guidelines

- Use React functional components.
- Use React hooks when required.
- Write clean and readable code.
- Use meaningful names for variables, functions, and components.
- Avoid unnecessary complexity.
- Follow modern React best practices.
- Prioritize maintainable and scalable solutions.

## UI/UX Guidelines

- Build responsive designs for desktop, tablet, and mobile.
- Focus on a clean and professional user experience.
- Maintain consistent spacing, typography, and design patterns.
- Create reusable UI components.

## Git Commit Convention

Use Conventional Commits format.

Examples:

- feat: add new features
- fix: fix bugs
- docs: update documentation
- style: improve formatting or styling
- refactor: improve existing code
- chore: update configuration files

## AI Assistant Guidelines

- Understand the existing project structure before making changes.
- Explain major changes before implementation.
- Suggest improvements when appropriate.
- Avoid generating unnecessary code.
- Prefer simple and maintainable solutions.
- Follow the project's stack and coding conventions.

## Project Goal

Build a production-quality AI Career Companion application that helps users with:

- Resume creation and improvement.
- Cover letter generation.
- Interview preparation.
- Job application tracking.
- AI-powered career guidance.


## Framework Rules:

- This project must use React + Vite.
- Do not migrate to Next.js, Create React App, or other frameworks.
- Use JavaScript only unless explicitly requested.

## Component Rules:

- Use PascalCase for React component names.
- Keep one component per file.
- Create reusable components instead of repeating code.
- Keep components small and focused.

Examples:
- Navbar.jsx
- ResumeCard.jsx
- Dashboard.jsx

## Development Commands

Install dependencies:

npm install

Run development server:

npm run dev

Build project:

npm run build
